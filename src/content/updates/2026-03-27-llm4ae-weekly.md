This week we completed the first-stage pipeline training for a fine-tuned local BioBERT transformer model for clinical Named Entity Recognition (NER).

### Key Updates
- **BioBERT Model Ingestion:** Set up Conda profiles and training scripts to extract adverse events and drug terms.
- **Narrative Tagging:** Added support for character offset alignments when processing multi-sentence narrative text blocks.
- **Metadata Layouts:** Optimized sidebar component displays in React to show temporal links and categories.
