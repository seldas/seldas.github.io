**HetEnc** (Heterogeneous Autoencoder) is a deep learning architecture designed for high-dimensional biological data integration. It focuses on extracting robust low-dimensional feature representations from complex gene expression datasets to improve predictive modeling.

### Key Features
- **Heterogeneous Autoencoders:** Novel autoencoder structure optimized for processing diverse genomic data layers.
- **Deep Feature Representation:** Compresses genomic features while preserving critical biological signals.
- **MLP Predictors:** Integrates multi-layer perceptron models for downstream classification and survival analysis based on encoded features.
