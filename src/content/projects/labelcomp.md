**LabelComp** is a specialized analytical tool designed to assist drug reviewers, pharmacovigilance experts, and researchers in tracking changes in FDA-approved drug labeling. 

### Key Features
- **Semantic Differencing:** Uses clinical NLP models to compare different versions of labeling documents (specifically safety warnings, precautions, and adverse reactions sections).
- **Adverse Event Extraction:** Leverages AI to extract standardized adverse event (AE) terms and highlight new safety signals.
- **Visual Comparison:** Provides reviewers with a clear visual diff of updates between document versions, minimizing manual search.
